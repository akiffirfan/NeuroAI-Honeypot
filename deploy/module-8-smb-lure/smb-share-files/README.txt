Neuro AI Dataset Repository
===========================

Access: \\neuro-train-01\neuro-data-share

This is the primary dataset storage share for the Neuro ML training platform.
All training data, model manifests, and workspace exports are stored here.

READ-ONLY for non-service accounts.
Request write access from: priya.nair@neuro.ai (IT ticket required)

Datasets are organized by project:
  /datasets/         — training datasets (ImageNet, WIT, internal)
  /models/           — model checkpoints and manifests
  /workspaces/       — per-team workspace exports
  /configs/          — production training configurations

For questions contact the ML Infrastructure team (Slack: #ml-infra).
