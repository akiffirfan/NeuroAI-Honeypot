Authentication

All protected endpoints require a Bearer token:

Authorization: Bearer <token>

Obtain a token via POST /api/v1/auth or via Microsoft SSO (GET /auth/sso). Tokens expire after 8 hours.

Note: Several internal endpoints are currently missing auth middleware — tracked in IT#5117. Fix scheduled for next sprint.
